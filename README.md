
# 🎂 Event System

Different event systems made in C++ for learning purposes, event systems are made with useability and performance in mind. Contains an event dispatcher along with RAII based handler for automatic clean up of events.  
Contains several projects for different varients of event systems.

## ✨ Features

- Event Interface
- Examples of different types of events implemented
- Event dispatcher singleton
- RAII based event handler class
- Easy function binding system
- Type safe event system project similar to unreal's event system

## 🛠️ Built With

- **Language**: C++
- **Build System**: [CMake](https://cmake.org/)
- **Platform**: Windows

## 📜 Instructions

- Clone repository
- Open `CMD` and run `cmake -B build` to generate solution file
